# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

`interactive-articles` is the publication surface for **Pixels to Motors / Motion UX** articles and interactive demos. Each top-level folder is one article (HTML + assets), aggregated by `articles.json` and surfaced via `index.html`.

## Motion UX 戦略の参照ルール

このリポジトリには `STRATEGY.md`（Motion UX コンテンツ戦略ドキュメント）が置かれている。以下の場合は必ず最初に `STRATEGY.md` を読むこと：

- 「次に何の記事を書くべきか」と聞かれた時
- 「進捗を確認して」と聞かれた時
- 新しい記事フォルダを作る時（カテゴリ ①〜⑤ のどれに属するか確認）
- 「Motion UX 戦略を更新して」と頼まれた時

進捗チェックリストの更新方法：
- 記事や動画が完成したら、対応する `- [ ]` を `- [x]` に変更
- 完成日を脇に併記する（例：`- [x] #1 Motion UX とは何か（決定版） — 2026-06-15`）
- 更新後は必ず「改訂履歴」セクションに1行追記する
